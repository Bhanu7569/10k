Documentation of CSS Selectors

  1. Element selector :
      2. In CSS, an element selector targets and styles HTML elements based on their tag names.
  2. Universal selector :
      3. In CSS, the universal selector (*) selects and styles all elements on a webpage.
  3. Group selector :
      4. In CSS, a group selector combines multiple selectors to apply the same style rules to multiple elements.
  4. ID selector :
      5. In CSS, an ID selector styles a specific HTML element with a unique identifier assigned using the `id` attribute.
  5. Class selector :
      6. In CSS, a class selector targets and styles HTML elements with a specific class attribute.
  6. Descendent selector
      7. In CSS, a descendant selector selects an element that is a descendant of another specified element.
  7. Child selector
      8. In CSS, a child selector targets and styles elements that are direct children of a specified parent element.
